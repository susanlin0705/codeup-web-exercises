What is JQuery?

jQuery is a cross-platform JavaScript library designed to simplify the client-side scripting of HTML. It was released in January 2006 at BarCamp NYC by John Resig.

Why use jQuery?

  • navigate a document
  • select DOM elements
  • create animations
  • handle events
  • create Ajax applications
  • add plugins
